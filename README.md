# ResumeSight — Resume analysis + eye-tracking MVP

ResumeSight is a minimal viable product that demonstrates an end-to-end resume analysis experience with optional eye-tracking signals to help improve recruiter engagement. The current workspace contains a static frontend and a working Python fallback backend for rapid development; a Spring Boot backend is present in the project but may require Maven to build and run.

Core idea
- Upload a resume (PDF / DOCX / TXT) and optionally paste a Job Description (JD).
- The frontend can collect eye-tracking points via WebGazer (when the browser can load the required models).
- The backend parses the resume, computes an ATS-style score, extracts skills/sections, compares against the JD, and returns recommendations. When gaze points are available they are analyzed for simple engagement insights.

Key components
- Frontend: `frontend/index.html`, `frontend/style.css` — static single-page interface with resume upload, JD textarea, and WebGazer integration.
- Backend (fast dev): `backend/server.py` — Python Flask API (`/api/analyze`) that extracts resume text, scores the resume, and returns structured recommendations.
- Backend (Java): `backend/src/main/java/com/resumesight/Application.java` and `pom.xml` — Spring Boot code exists but requires Maven to build & run.

Features
- Resume parsing: PDF/DOCX/TXT text extraction.
- ATS scoring: skills detection, section completeness, action verbs, quant metrics.
- JD matching: keyword & skill overlap scoring plus JD-specific recommendations.
- Gaze analysis: basic engagement summary when gaze data is provided.

Run locally (quick)
1. Start the frontend (serves static files):

```powershell
cd frontend
python -m http.server 8000
# open http://localhost:8000 in your browser
```

2. Start the Python backend (if you prefer the fast dev server):

```powershell
cd backend
# create and activate a venv if desired
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt  # or: pip install flask python-docx PyPDF2 flask-cors
python server.py
# backend runs on http://localhost:8080
```

Notes about the Spring Boot backend
- If you want to run the Java Spring Boot backend, ensure Maven is installed and run from the `backend` directory:

```powershell
cd backend
mvn spring-boot:run
```
